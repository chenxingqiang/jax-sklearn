## Usage

```python
from secret_learn.algorithms import algorithms

# FL mode - data stays in local PYUs
model = algorithms.FLSGDClassifier(
    devices={'alice': alice, 'bob': bob},
    heu=heu
)

model.fit(fed_X, fed_y, epochs=10)
```
